INSTALL
1. Stop bot: Ctrl+C
2. Extract ZIP into a new folder.
3. Copy .env, marketplace.db and facebook-profile from old project.
4. Run: py -m pip install -r requirements.txt
5. Test: py -m unittest discover tests
6. Start: py app.py
